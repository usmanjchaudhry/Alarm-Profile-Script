README
--------------------------------------------------
Script created to clean and display monthly Alarm Profile data.

Instructions for use:

1.alarmProfile application must be in the same directory as the files of data.

2.Data must follow naming convention: 'Alarm profile for...' for program to work correctly. (Ex: Alarm Profile for Aug 29)

3.Data must be xlsx files cannot be csv files.

4.There must be an output folder in the same directory as the application. Output folder must be titled: 'outputFolder'
